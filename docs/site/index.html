Nacre Documentation

Nacre compiles a compact script-oriented language into standalone Bash. This site covers the implemented compiler, tutorial flow, language reference, CLI, self-compilation, and coverage gates.

Read tutorial Language reference 
$ cargo run -- docs/examples/hello.ncr
#!/usr/bin/env bash
set -euo pipefail

readonly name='Nacre'

echo Hello from ${name} || exit $?

Start

Install

Use the Rust toolchain described by rust-toolchain.toml.

cargo test

Compile

Write a .ncr file and compile it to Bash.

cargo run -- docs/examples/hello.ncr /tmp/hello.sh

Run

The generated script is a standalone Bash file.

bash /tmp/hello.sh

Implemented Surface

Modules

use imports functions, top-level bindings, types, newtypes, and traits under the final path component. Declarations starting with _ stay private to their module.

Types

Primitive values, arrays, maps, records, type aliases, newtypes, options, results, marker traits, impl methods, and bounded generic functions are checked before Bash emission.

Strings and Paths

Method calls cover common script operations such as slice, trimStart, trimEnd, repeat, isAbsolute, basename, dirname, stem, and extname.

Control Flow

Statement-level if, else if, else, bare blocks, while, for, break, and continue are emitted as Bash control structures.
